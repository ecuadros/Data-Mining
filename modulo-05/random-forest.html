Random Forest

Ensamble de árboles de decisión entrenados con bootstrap sampling y selección aleatoria de features. Haz click en el canvas para clasificar un punto — observa los votos individuales de cada árbol.

Idea central

Un solo árbol sobreajusta. Si entrenamos muchos árboles sobre muestras bootstrap diferentes y promediamos sus votos, la varianza cae sin aumentar el sesgo.

ŷ = argmax_c Σᵢ 𝟙[hᵢ(x)=c]

Dos fuentes de aleatoriedad

  • Bootstrap: cada árbol ve ~63% de los datos (con reemplazo)
  • Feature subsampling: en cada split solo se considera √p features aleatorios

Esto descorrelaciona los árboles → la reducción de varianza es real.

Out-of-Bag (OOB) error

El ~37% de instancias no vistas por cada árbol sirve como validación gratuita. El OOB error estima el error de generalización sin necesidad de un conjunto de test separado.
