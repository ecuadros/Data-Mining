Árbol de Decisión

Un árbol que divide el espacio de features recursivamente para separar las clases. Cada nodo interno es una pregunta; cada hoja, una predicción.

Algoritmo ID3 / C4.5 / CART

  1. Si todos los ejemplos son de la misma clase → hoja
  2. Elegir el feature que más reduce la impureza (Gini o Entropía)
  3. Dividir el dataset por el umbral óptimo
  4. Repetir recursivamente en cada subconjunto

Criterio de división (CART)

Gini(S) = 1 − Σ pᵢ²

La ganancia de información de un split mide cuánto se reduce la impureza del nodo padre al dividir.

Ganancia = Impureza(padre) − Σ (|Sᵢ|/|S|)·Impureza(Sᵢ)

Control del sobreajuste

  • Max profundidad: limita cuántos niveles puede crecer
  • Min muestras por hoja: evita nodos con muy pocos datos
  • Poda (pruning): elimina ramas que no mejoran el test set
