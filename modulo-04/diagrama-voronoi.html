Diagrama de Voronoi

Cada región contiene todos los puntos del plano más cercanos a un centroide. Es exactamente la frontera de decisión de 1-NN. Haz click para añadir centroides, arrástralos para moverlos.

Definición

Dado un conjunto de sitios (centroides) S = {s₁…sₙ}, la región de Voronoi de sᵢ es:

V(sᵢ) = { p : dist(p,sᵢ) ≤ dist(p,sⱼ) ∀j≠i }

Las fronteras son segmentos de mediatriz entre pares de sitios adyacentes.

Relación con KNN

  • El diagrama de Voronoi = frontera de decisión de 1-NN
  • Para k>1, las fronteras se suavizan (el demo KNN lo muestra)
  • Triangulación de Delaunay: dual geométrico del diagrama

Aplicaciones en Minería de Datos

  • Análisis de zonas de influencia (tiendas, hospitales)
  • Segmentación geográfica de clientes
  • Interpolación espacial de sensores
  • Análisis de redes de distribución
