Explorador de Datasets

Antes de aplicar cualquier algoritmo, es esencial inspeccionar el dataset: sus dimensiones, tipos de atributos, balance de clases y valores faltantes. Selecciona uno de los tres datasets clásicos y explora sus características al instante.

¿Qué es un Dataset de DM?
  • Instancias Cada fila es un objeto, ejemplo o registro individual.
  • Atributos Cada columna describe una característica (numérica, nominal, ordinal…).
  • Clase (target) Atributo especial que el modelo intenta predecir.
Formatos comunes
CSV ARFF JSON Parquet HDF5

ARFF (Attribute-Relation File Format) es el formato nativo de Weka. CSV es el más universal. Parquet es columnar y eficiente para Big Data.

Métricas de inspección
  • Dimensiones n instancias × p atributos.
  • Tipos de atributos Numéricos vs nominales vs binarios.
  • Balance de clases ¿Están las clases representadas por igual?
  • Valores faltantes % de NaN por atributo — afecta el preprocesamiento.
