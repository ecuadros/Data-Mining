Selección de Features Avanzada

Compara 4 métodos de ranking de features sobre un dataset de churn. Selecciona features manualmente y observa cómo cambia la precisión de KNN (k=3) y la proyección 2D.

Métodos de selección

  • Varianza: descarta features con poca variación (casi constantes)
  • Correlación con target: Pearson r entre feature y clase
  • Información mutua: aprox. por discretización en bins
  • RFE score: importancia estimada por eliminación recursiva
