Validación Cruzada K-Fold

La validación cruzada estima el error de generalización de forma honesta. Cada fold actúa como test una vez. Compara con el error in-sample (train=test) para ver el sobreajuste.

Procedimiento

  1. Dividir datos en k folds iguales
  2. Para cada fold: entrenar con k-1, evaluar en el fold restante
  3. Promediar los k errores → estimación imparcial
