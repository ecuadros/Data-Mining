Matriz de Correlaciones

Explora las relaciones lineales entre atributos usando el coeficiente de Pearson y visualización heatmap.

📐 Fundamentos Teóricos

Coeficiente de Pearson

Mide la fuerza y dirección de la relación lineal entre dos variables cuantitativas.

r = Σ((xᵢ − x̄)(yᵢ − ȳ)) / (n · σx · σy)
r ∈ [−1, +1]

Interpretación

  • r ≈ +1 → correlación positiva fuerte
  • r ≈ +0.5 → correlación positiva moderada
  • r ≈ 0 → sin correlación lineal
  • r ≈ −0.5 → correlación negativa moderada
  • r ≈ −1 → correlación negativa fuerte
Correlación ≠ Causalidad: El número de películas de Nicolas Cage estrenadas por año correlaciona con ahogamientos en piscinas (r ≈ 0.67). Evidentemente, Cage no causa ahogamientos — ambas suben en verano.

Uso en Minería de Datos

La matriz de correlaciones permite:

• Selección de features: eliminar atributos redundantes (alta correlación entre sí).

• Detección de multicolinealidad antes de aplicar regresión.

• Identificar predictores potentes para la variable objetivo.

⚠️ Limitación importante: Pearson solo detecta relaciones lineales. Dos variables pueden tener r ≈ 0 y aún así estar fuertemente relacionadas de forma no lineal (p. ej. cuadrática, cíclica).
